# BACKROOM LP

XServer Static公開用の静的サイトです。

## ファイル
- `index.html`
- `styles.css`
- `script.js`

## GitHubへのアップロード
1. GitHubの `backroom` リポジトリを開く
2. `Add file` → `Upload files`
3. この3ファイルをドラッグ＆ドロップ
4. `Commit changes`
5. XServer Staticが自動デプロイするまで数分待つ
